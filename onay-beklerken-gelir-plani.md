# Onay Beklerken Gelir Plani

Pinterest Standard Access beklerken para akisi icin API kritik yol degil. Kritik yol: kaliteli manuel pin, owned guide sayfalari, gercek affiliate/odeme linkleri ve takip.

## Bugun Yapilacaklar

1. `npm run manual:pack` ile manuel yukleme paketi uret.
2. `exports/manual-upload/<tarih>/manual_upload_pins.csv` dosyasindaki ilk 3-5 pini Pinterest arayuzunden elle yukle.
3. Her pini ilgili board'a yukle, linki owned guide/checklist sayfasina ver.
4. Direkt affiliate link kullanma. Affiliate program onayi geldikten sonra guide sayfasindaki ilgili urun kartlarina gercek link ekle.
5. 24-72 saat sonra Pinterest Business Analytics'te impression, outbound click ve save verisini kontrol et.

## Para Kazanma Sirasi

1. Owned digital product: en hizli kontrol edilebilir model. Stripe, Payhip veya Gumroad payment link gerekir.
2. Affiliate: program onayi ve gercek takip linki gerekir. Link yoksa komisyon da yoktur.
3. Display ads: AdSense/Ezoic gibi aglar icin once kaliteli site, politika uyumu ve trafik gerekir.
4. Sponsorlu yerlesim: trafik ve niche kaniti olusmadan satmasi zordur.

## Affiliate Program Onceligi

1. Hostinger referral: aktif link `https://www.hostinger.com?REFERRALCODE=HAJAYIYIGB9F`; site/hosting intent'i olan iceriklere uygundur.
2. Fiverr Affiliates: logo, web sitesi, freelancer ve business rehberlerine uygundur.
3. Impact / PartnerStack uzerindeki SaaS programlari: AI tools ve productivity rehberleri icin uygundur.
4. Amazon Associates: home, beauty, meal prep ve organizer urunleri icin uygundur; ulke ve sosyal medya/site kosullari hesap acilirken tekrar kontrol edilmelidir.
5. CJ: opsiyonel. Giris sorunu devam ediyorsa kritik yol degil.

## Guvenlik Kurallari

- Placeholder affiliate URL yayinlama.
- Link shortener kullanma.
- Direkt affiliate Pin'i toplu yayinlama.
- Ticari link varsa acik disclosure kullan.
- Kopya urun gorseli ve abartili gelir/saglik iddiasi kullanma.
- API onayi gelene kadar otomatik publish calistirma.
